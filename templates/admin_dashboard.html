포스터 관리

키오스크에 표시될 포스터를 관리합니다. 새 포스터를 업로드하거나 기존 포스터를 삭제할 수 있습니다.

  • 업로드 가능한 이미지 형식: JPG, PNG, GIF
  • 최대 파일 크기: 16MB
  • 권장 이미지 비율: A4 (595 x 842 픽셀)

키오스크 설정

키오스크의 기본 정보를 설정합니다. 제목, 연락처, 운영 시간 등을 변경할 수 있습니다.

비밀번호 변경

관리자 계정의 비밀번호를 변경합니다. 보안을 위해 주기적으로 비밀번호를 변경하는 것이 좋습니다.

비밀번호는 최소 6자 이상이어야 합니다.

급식 관리

급식 정보를 관리하고 설정합니다. 나이스 API 키, 학교 코드, 교육청 코드 등을 입력하여 급식 정보를 자동으로 가져올 수 있습니다.

자동 갱신을 활성화하면 설정한 시간에 급식 정보가 자동으로 업데이트됩니다.
